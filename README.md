# Sales Performance & Customer Analytics Dashboard

## Project Overview

This project analyzes sales and customer data using Microsoft Excel. The goal was to identify sales trends, top-performing products, profitable regions, and customer purchasing behavior through an interactive dashboard.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* XLOOKUP
* VLOOKUP
* INDEX-MATCH
* SUMIFS
* COUNTIFS
* Conditional Formatting

## Key Features

* Data Cleaning and Preparation
* KPI Cards (Total Sales, Profit, Orders, Average Sales)
* Region-wise Sales Analysis
* Product Category Analysis
* Monthly Sales Trend Analysis
* Top Products Analysis
* Interactive Dashboard using Slicers

## Business Insights

* Identified top-performing sales regions.
* Analyzed category-wise revenue contribution.
* Tracked monthly sales trends.
* Highlighted top-selling products.
